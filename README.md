# Sudoku Solver (Pygame)

A GUI-based Sudoku solver built using Python and Pygame.

## Features
- Interactive Sudoku board
- Backtracking solver
- Visual solving animation
- Timer and mistake counter

## Controls
- Click: Select cell
- Number keys: Input value
- Enter: Confirm value
- Space: Auto solve
- Delete: Clear cell

## Installation
pip install pygame

## Run
python main.py
